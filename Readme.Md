# 🏥 DocLink – Doctor Appointment Platform

DocLink is a full-stack doctor appointment booking platform that connects patients, doctors, and administrators in a single application. It allows patients to book appointments seamlessly, doctors to manage their schedules, and admins to manage the overall platform efficiently.

## 🚀 Features

### 👤 Authentication
- JWT Authentication
- Google OAuth Login
- Secure Password Hashing
- Role-Based Access Control (Patient, Doctor, Admin)

### 🩺 Patient
- Browse available doctors
- Book conflict-free appointments
- View appointment history
- Manage profile

### 👨‍⚕️ Doctor
- Manage appointment schedule
- View patient appointments
- Update appointment status
- Manage profile

### 🛠️ Admin
- Manage doctors
- Manage users
- Monitor appointments
- Platform administration
